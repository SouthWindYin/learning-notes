# java查看字节码

使用命令`javap -c XXX.class`

## 例

分析自动拆箱装箱机制的程序

``` java
public class Demo {

	public static void main(String[] args) {
		int a=128;
		int b=128;
		
		int c=3;
		int d=3;
		
		boolean eq1=(a==b);
		boolean eq2=(c==d);
	}
}
```

字节码输出

``` cmd
javap -c Demo.class
```
